# IKF Kenshido Tournament Management System

A professional tournament management system built with Next.js for managing martial arts competitions.

## Features

- **Tournament Dashboard** - Real-time tournament overview and management
- **Athlete Management** - Register and manage athlete profiles
- **Club Management** - Organize and track club registrations
- **Bracket System** - Automated bracket generation and management
- **Judging Interface** - Dedicated interfaces for judges and chief judges
- **Referee Management** - Track and assign referees
- **Weigh-in System** - Manage athlete weigh-ins
- **Round Management** - Control tournament rounds and progression
- **Live TV Display** - Public-facing displays for spectators
- **Statistics & Reports** - Comprehensive analytics and reporting
- **AI Assistant** - Built-in AI helper for tournament management

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Tech Stack

- **Framework:** Next.js 14 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Database:** Firebase/Firestore
- **Real-time:** Socket.io
- **UI Components:** Custom IKF-branded components

## Project Structure

```
/app              - Next.js app router pages
/components       - Reusable UI components
/lib              - Utility functions and integrations
/store            - State management stores
/types            - TypeScript type definitions
```
